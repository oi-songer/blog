Codeforces Gym 101804H

题目来源: Codeforces

分析

这题给出了一个图，给出q次询问，要求每次询问中分为从点1到某点的途经城市数小于某个值的最短路。我们只需要让原来存储距离的dist[]变为dist[][]即可，一维存放的是目的地，另一维存放的是走了多少个城市。然后，在SPFA()运行完时递推地传递一下值，令$dist_{i,j} = min( dist_{i,k} ) , k $即可。
