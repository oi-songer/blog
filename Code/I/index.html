I'm Telling the Truth

hdu 3729

题目来源: hdu

分析

这题第一眼看上去或许不太像二分图匹配问题。但是我们仔细研究一下便可以发现，我们可以把学生和分数当作二分图的两个部分，因为分数小于\(100000\)，所以我们可以直接开一个数组存储它的belong[]。然后就二分图匹配，找到匹配的数量，便是我们要的"说真话的学生数量"。
